## Patterns Established

1. **Strategy map pattern:** Framework capabilities are expressed as a flat `OUTPUT_STRATEGIES` object keyed by framework identifier (e.g., `'next'`, `'vue'`). Each entry is a plain object with a consistent schema. New frameworks are added by extending the map — no conditional branching required at the call site.

2. **Graceful fallback to HTML:** Any unknown, null, or empty framework identifier silently falls back to the `html` strategy. This is the project-wide default. The fallback chain is: `(framework || 'html').toLowerCase()` then `OUTPUT_STRATEGIES[key] || OUTPUT_STRATEGIES.html`.

3. **Non-destructive projectConfig overrides:** `getOutputStrategy` returns a shallow copy (`{ ...base }`) and applies overrides on top. The original `OUTPUT_STRATEGIES` map is never mutated. Config values (cssFramework, language, routerType) drive selective overrides rather than wholesale replacements.

4. **`{slug}` template placeholder convention:** Strategy `outputPath` fields use `{slug}` as a placeholder (e.g., `app/articles/{slug}/page.tsx`). Resolution is always via `resolveOutputPath(strategy, slug)` which calls `.replace(/\{slug\}/g, slug)`. This pattern is consistent across all 7 strategies.

5. **editOverlay two-value enum:** Edit overlay type is encoded as one of two string values: `'client-component'` (React/Next.js — requires `'use client'` directive) or `'vanilla-js'` (all other frameworks — inline script tag). The value is read directly from `strategy.editOverlay` with no further logic needed.

6. **Framework router as pipeline pre-step:** The router is invoked via a `node -e` bash snippet before the draft-writer agent is dispatched. The resolved `{ strategy, slug, outputPath }` object is passed into the agent prompt as context. The agent never re-resolves the strategy — it consumes the pre-resolved values.

7. **node:test with no external dependencies:** Test file uses only `node:test` and `node:assert/strict` — zero npm packages required. Consistent with the project's zero-dependency philosophy.
