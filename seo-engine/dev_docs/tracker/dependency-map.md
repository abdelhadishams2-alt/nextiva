## Critical Path

The longest chain of hard dependencies that determines minimum project duration:

```
DI-001 (OAuth, 16h, S1)
  -> DI-002 (GSC, 12h, S2)
    -> DI-005 (Scheduler, 12h, S3)
      -> CI-001 (Decay, 12h, S4)
        -> CI-003 (Recommender, 16h, S5)
          -> CI-005 (Dashboard: Opportunities, 12h, S6)
            -> [wait for VI-001/VI-002]
              -> VI-003 (Voice Agent, 12h, S7)
                -> [wait for PB-001]
                  -> PB-002 (WordPress, 20h, S8)
                    -> PB-004 (CMS Adapters, 12h, S9)
                      -> FL-001 (Tracker, 12h, S10)
```

**Length:** 10 tasks on critical path
**Estimated Hours:** ~136h (sum of critical-path task efforts)
**Span:** Sprints 1-10 (entire 20-week project)
