## Stack Conventions

### Backend (Bridge Server)
- Pure Node.js — `http.createServer()`, no Express, no npm packages
- All async I/O — never use `readFileSync` in handlers
- JSON response envelope: `{ success: true, data }` or `{ error, message }`
- Auth: `verifyAuth(req)` returns user object or null
- Logging: `createLogger('module-name')` from bridge/logger.js

### Database (Supabase)
- REST client via `supabase-client.js` (native fetch) — no ORM
- All tables: UUID PKs, `user_id` FK, `created_at`/`updated_at`
- RLS on every table: `auth.uid() = user_id`
- Encrypted fields: AES-256-GCM via `bridge/key-manager.js`

### Frontend (Dashboard)
- Next.js 16 App Router with React Server Components
- shadcn/ui + Tailwind CSS (dark mode default, zinc + gold accents)
- Server components for initial data fetch, client components for interactivity
- `fetch()` with ISR caching (`next: { revalidate: N }`)

### Testing
- `node:test` (built-in) — no Jest, no Mocha
- Pattern: `describe` → `it` → `assert.equal` / `assert.ok` / `assert.rejects`
- Test files: `test/*.test.js` (unit), `test/integration/*.test.js` (integration)
- Run: `node --test test/` or `npm test`
