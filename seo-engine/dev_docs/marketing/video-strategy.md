## LinkedIn Native Video Strategy

### Why Native (Not YouTube Links)
- LinkedIn suppresses posts with external links (50-70% less reach)
- Native video autoplays in feed — higher engagement
- Shorter format fits LinkedIn consumption patterns

### LinkedIn Video Specs
| Spec | Value |
|------|-------|
| Length | 60-90 seconds (max 2 min) |
| Format | MP4 |
| Aspect ratio | 1:1 (square) or 4:5 (vertical) — both outperform 16:9 |
| Resolution | 1080x1080 or 1080x1350 |
| Captions | Burned in (not relying on platform captions) |
| File size | Under 200MB |

### LinkedIn Video Content
- 60-second cuts of full YouTube demos
- Quick tips (single feature highlight)
- Testimonial soundbites (best 60 seconds)
- "Did you know?" format for educational clips
- Behind-the-scenes building montages

## Video Production Workflow

### Per-Video Process (Bootstrap Phase)
1. **Outline** (30 min): Script bullet points, key screens to show
2. **Screen recording** (45 min): Record in OBS, multiple takes if needed
3. **Voiceover** (30 min): Record separately in Audacity for cleaner audio
4. **Editing** (60 min): Sync audio/video in DaVinci Resolve, add transitions, trim dead time
5. **Thumbnail** (15 min): Create in Canva from template
6. **Subtitles** (30 min): Upload to YouTube, correct auto-captions, add Arabic track
7. **Publish** (15 min): Title, description, tags, end screen, cards, schedule
8. **Promote** (15 min): LinkedIn post, Twitter link, email mention

**Total per video: ~4 hours**
**Monthly time investment (2 videos): ~8 hours**
