## Tool Capabilities

### node:test (Built-in)
- `describe` / `it` / `before` / `after` / `beforeEach` / `afterEach`
- `assert.equal` / `assert.ok` / `assert.rejects` / `assert.deepEqual`
- Native `--test` flag: `node --test tests/**/*.test.js`
- Parallel execution: `node --test --test-concurrency=4`
- TAP output: `node --test --test-reporter=tap`

### Playwright MCP (via Claude Code Plugin)
Available MCP tools:
- `browser_navigate` — open URL
- `browser_click` — click element by ref
- `browser_fill_form` — fill form fields
- `browser_take_screenshot` — capture viewport
- `browser_snapshot` — get accessibility tree
- `browser_console_messages` — read console output
- `browser_network_requests` — monitor API calls
- `browser_evaluate` — run JS in page context
- `browser_wait_for` — wait for selector/network

### ESLint + TypeScript
- `npx next lint` — runs ESLint on dashboard
- `npx tsc --noEmit` — type-checks without emitting
- Both run in CI pipeline (`.github/workflows/ci.yml`)
