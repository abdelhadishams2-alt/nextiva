1 Data Ingestion & Validation GSC API (OAuth2), Screaming Frog API or
direct crawler, CMS REST feed. Handles encoding, dedup, and URL
canonicalisation. Depends: GSC access, CMS API credentials

2 NLP Entity Extraction Arabic-tuned NLP models for NER, topic
extraction, and relationship mapping. Will build per-article entity
vectors stored in vector DB. Depends: Module 01 output

3 Scoring & Ranking Engine Composite authority score: backlink strength
(Ahrefs/Majestic API), engagement signals (GA4), recency decay function,
topical relevance (cosine similarity). Depends: Modules 01--02, Ahrefs
API

4 Semantic Clustering HDBSCAN clustering on page embedding vectors. Will
identify topic clusters, orphan pages, and competitive overlap. Outputs
cluster map and cannibalization risk matrix. Depends: Module 02 vectors

5 Consolidation Engine Rule-based + ML decision system for
merge/redirect recommendations. Will evaluate authority transfer
projections per consolidation action. Depends: Modules 03--04

6 Dashboards & Reporting 4 dashboards in Looker Studio: Traffic
Authority, Cannibalization Matrix, Opportunity Backlog, Internal Link
Equity. Refreshed on GSC data cycle (24--48hr). Depends: Modules 01--05
