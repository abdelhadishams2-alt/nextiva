# Shared UI Components

## Article Components
- **TOCSidebar** (src/components/ui/TOCSidebar.tsx) — Sticky sidebar TOC with IntersectionObserver
- **TOCInline** (src/components/ui/TOCInline.tsx) — Collapsible mobile TOC
- **ReadingProgress** (src/components/ui/ReadingProgress.tsx) — Fixed top progress bar
- **ShareButtons** (src/components/ui/ShareButtons.tsx) — Twitter/X, LinkedIn, Copy Link
- **FadeUpObserver** (src/components/ui/FadeUpObserver.tsx) — IntersectionObserver for .fade-up animations
- **AffiliateDisclosure** (src/components/ui/AffiliateDisclosure.tsx) — Disclaimer box
- **AffiliateSidebar** (src/components/ui/AffiliateSidebar.tsx) — Featured tool in sidebar
- **AffiliateMobileBar** (src/components/ui/AffiliateMobileBar.tsx) — Sticky mobile CTA bar
- **AffiliateMidArticle** (src/components/ui/AffiliateMidArticle.tsx) — Mid-article CTA card

## Section Components
- **Navbar** (src/components/sections/Navbar.tsx) — Mega menu with mobile accordion
- **CallToAction** (src/components/sections/CallToAction.tsx) — CTA section with bg image
